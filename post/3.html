图片和路径

图标标签

在html中标签中标签用来定义html中图像

src是img标签中必须属性，它用来指定图像的路径和文件名

常见属性：

1、src 图像属性 必须属性

2、title 文本 指示文本，当鼠标放在图像上显示的文字

3、alt 文本 替换文本，当图像不能显示时出现的文本

4、width/height

​ 如果用户不指定宽和高，则按照原来的图片显示

​ 如果用户指定宽和高，则按照用户指定的大小来显示

​ 指定图片只指定宽度和高度一个即可，另一个会自动缩放

图像：

图像类型：在网页中显示图像类型还固定

-jpg/jpeg 色彩丰富

-png 色彩效果最好

-gif 色彩单调-显示简单动画

-webp 谷歌公司推出的图像类型–小-颜色丰富-动画 兼容性差

如何下载适合小图标：https://www.iconfont.cn

图像路径：

​ 1、相对路径：本服务器资源使用

​ —兄弟资源 ：直接写资源名称 src=”小米.png”

​ —子资源：先写目录名在写资源名称 src = “img/小米.png”

​ —父资源：先../找父元素，在写资源名称 scr=”../小米.png”

​ —绝对路径： src=”http://www.baidu.com/xx.png“

超链接

意义：从一个网页跳转到另外一个网页标签

语法：文字

​ href=”” 跳转链接网页的网页地址

​ target=””

​ _self 默认值 在当前页面打开新链接网页

​ _blank 在新页面打开链接网页

超链接的其他形式

1、下载文件 href=”1.zip”

2、发送邮件 href=”mailto:邮箱名”

3、返回页面顶部 href=”#”

4、使用js阻止跳转 href=”javascript:;”

锚点

在网页中定义一个记号（锚点）点击A标签时，跳到这个记号的位置

1、定义锚点 <a name=”锚点名称”> name 定义锚点

2、创建超链接跳转锚点

特点：整齐，整洁，有序，它作为布局会更加自由方便

无序列表，有序列表，自定义列表
